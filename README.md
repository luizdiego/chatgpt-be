# chatgpt-be

## Objective

Build a backend using **Next.js** that will serve as the foundation for a ChatGPT interface. The project will evolve incrementally, with this README updated as new features and components are added throughout development.

## Planned Scope

- Next.js backend (API routes)
- Integration with the OpenAI ChatGPT API
- Frontend chat interface (to be added in a later phase)

> Work in progress — this document will be updated as the project grows.
